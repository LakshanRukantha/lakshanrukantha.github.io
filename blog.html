Windows 11 Operating System

History of Windows

Windows මෙහෙයුම් පද්ධතිය පිළිබඳව කතා කිරීමට පෙර අප Microsoft සමාගම පිළිබඳව දැන සිටීම වැදගත් වේ. Microsoft යනු 1975 වර්ෂයේ Bill Gates හා Paul Allen විසින් ආරම්භ කරන ලද සමාගමකි. මෙහි Microsoft යන නාමය පිළිබඳව සලකා බලන විට එය Microcomputer Software යන වචන වල එකතුවෙන් සකස් කරන ලද්දකි.

Windows හි මුල්ම මෙහෙයුම් පද්ධතිය එළිදක්වන ලද්දේ Bill Gates විසින් වර්ෂ 1985 දී ය. එය Windows 1 ලෙස නම් කරන ලදී. එදා පටන් මේ දක්වා Windows මෙහෙයුම් පද්ධතියේ විවිධ අනුවාදයන් එළිදක්වන ලදී. ඒවා එළිදැක්වූ වර්ෂ සමග පහත දැක්වේ.

  • Windows 2 – (1987)
  • Windows 3 – (1990)
  • Windows 3.1 – (1992)
  • Windows 95 – (1995)
  • Windows 98 – (1998)
  • Windows ME – (2000)
  • Windows XP – (2001)
  • Windows Vista – (2006)
  • Windows 7 – (2009)
  • Windows 8 – (2012)
  • Windows 10 – (2015)
  • Windows 11 – (2021)

Windows 11 update

වර්ෂ 2015 දී Windows 10 මෙහෙයුම් පද්ධතිය එළිදක්වන විට ඔවුන් පැවැසුයේ නැවත Windows හි අනුවාදයන් ලබා නොදෙන බවත් ඒ අවසාන මෙහෙයුම් පද්ධතිය වන බවට ය. කෙසේ නමුත් එම Microsoft ආයතනයේ එකල සිටි බොහෝ ඉහළ නිලධාරීන් දැන් වන විට එම ආයතනයේ නොසිටින බැවින් හා Windows 10 හි පැවැති ඒකාකාරී බැවින් පරිශිලකයන් වෙත Windows 11 මෙහෙයුම් පද්ධතිය පසුගිය 2021-06-24 දා නිළ වශයෙන් එළිදක්වන ලදී.

මෙම නව Windows 11 මෙහෙයුම් පද්ධතිය පිළිබඳ සලකා බලන විට එහි පරිශිලක අතුරු මුහුණත වැඩි දියුණු කිරීමට වැඩි මුලිකත්වයක් Microsoft ආයතනය විසින් ගෙන ඇති බව අපට පෙනී යයි. Windows 11 හි පැවැති Icons සියල්ලම පාහේ මෙම මෙහෙයුම් පද්ධතියේ අලුත් කර ඇත. වර්තමානයේ බොහෝ වෙබ් අඩවි සහ Mobile Applications බොහෝ ප්‍රමාණයක දැන් Light Mode සහ Dark Mode යොදා ගැනීමට පහසුකම් සලසා දී ඇති අතර මෙම Windows 11 මෙහෙයුම් පද්ධතියේ ද Light Mode සහ Dark Mode ලෙස ඔබ කැමති ලෙස සකසා ගත හැකිය.

Windows 11 හි වැඩිදියුණු කල Gaming වලදී සාමාන්‍ය graphics quality තත්වයකදී වුවද Auto HDR නම් අලුත් Feature එක මගින් Graphics වල තත්වය වඩාත් උද්දීපනය කිරීමට මොවුන් ක්‍රියා කර ඇත. තවද අප එදිනෙදා භාවිතා කරන Mobile Applications ඔබගේ නව Windows 11 මෙහෙයුම් පද්ධතිය තුළ භාවිත කළ හැකි වේ. තවත් ලබාදී ඇති නව Features මෙම Link එක හරහා ඔබට ලබාගත හැකිය.

Windows 11 requirements

අප වැඩි අවධානයක් යොමු කළ යුත්තේ බොහෝ දෙනා වර්තමානයේ Windows crack version තම පරිගනකයට Install කරගෙන භාවිතා කරයි. මෙම ආයතනය විසින් පවසන අන්දමට Windows 11 Update එකක් ලෙස ලබාගත හැක්කේ Original Windows 10 මෙහෙයුම් පද්ධතිය භාවිතා කරන අයට පමණි. එය මේ වසරේ අවසානයේදී බලාපොරොත්තු විය හැකිය.

මේ වන විට Windows 11 Leaked Versions අන්තර්ජාලය පුරා ඔබට පහසුවෙන් සොයාගත හැකි නමුත්. එම මෙහෙයුම් පද්ධතියක් ඔබගේ Original Windows මෙහෙයුම් පද්ධතිය මකා දමා ස්ථාපිත කලහොත් ඔබට පහසුවෙන් Windows 11 වෙත පෙර මෙහෙයුම් පද්ධතිය Update කිරිමේ හැකියාව නැති වී යනු ඇත. මෙම Windows 11 ඔබගේ පරිගණකය වෙත ස්ථාපනය කිරීමට සපුරාලිය යුතු සුදුසුකම් පහත දැක්වේ.

  • Processor : 1 gigahertz (GHz) or faster with 2 or more cores on a compatible 64-bit processor or System on a Chip (SoC)
  • RAM : 4 gigabyte (GB)
  • Storage : 64 GB or larger storage device
  • System firmware : UEFI, Secure Boot capable
  • TPM : Trusted Platform Module (TPM) version 2.0
  • Graphics card : Compatible with DirectX 12 or later with WDDM 2.0 driver
  • Display : High definition (720p) display that is greater than 9” diagonally, 8 bits per color channel
  • Internet connection & Microsoft accounts : Windows 11 Home edition requires internet connectivity and a Microsoft account to complete device setup on first use

What is Deep Fake?

වර්තමානය වන විට තාක්ෂණික දැනුම ලොකයේ හොඳ අතට මෙන්ම නරක අතට ද යොදාගන්නා බව අපට එදිනෙදා ලොකයේ සිදුවෙන සිදුවීම් ඔස්සේ මානව පැහැදිලි වෙයි. මුලින්ම මේ Deepfake කියන්නේ සරලවම “යම් පුද්ගලයකුගේ පින්තුරයක් හෝ Video පටයක් ඔහු සිදු නොකළ දෙයක් කෘතිමව නිර්මාණය කිරීමයි.” තවත් පැහැදිලි කරනවා නම් ඔබ විසින් නොකියු දෙයක් ඔබේම මුහුණේ ඉරියව් හා කටහඩ කෘතිම ලෙස සකසා යම් යම් ක්‍රියාවන් සඳහා යොදා ගැනීමයි.

History of Deep Fakes.

මෙහි ඉතිහාසය පිළිබඳ සලකා බලන විට මෙය මුල් වරට ආරම්භ වූයේ 1997 වර්ෂයේදී ය. එයට Machine Learning තාක්ෂණය යොදාගෙන තිබුණි. මෙම Deepfake Technology එක නිර්මාණය වන්නේ Artificial Intelligence හෙවත් කෘත්‍රීම බුද්ධිය යොදාගෙනයි.Artificial intelligence හෙවත් කෘතිම බුද්ධිය යොදාගෙන නිර්මාණය කරන ලද Artificial Nural Network හෙවත් කෘත්‍රීම ස්නායුක ජාලයකට වර්තමානය වන විට සතුන්ගේ මොලය තුළ සිදුවන සුළු වෙනස්කමක් පවා කියවීමට හැකියාව තිබේ. ඒවා අයත් වන්නේ Deep learning යන කොටසටයි. මෙය Deepfake සමග සම්බන්ධ වන්නේ මෙම Deep learning යොදාගෙන සැබෑ ලොකයේ සිටින පුද්ගලයකුගේ අංග චලන, මුහුනේ ඉරියව්, බාහිර පෙනුම හා කටහඩ යන සියලු දේ කෘතිම ලෙස විශේෂිත වූ පරිගණක පද්ධතියකට පුහුණු කිරීම හැකියාව තිබීමෙනි.

මොකක්ද මේ Cyber Security කියන්නේ?

අපි වර්තමානයේ ගොඩක් අන්තර්ජාලය භාවිත කරනවා. මේ දින වල Corona Pandemic එක නිසා බොහෝ ආයතන තමන්ගේ කාර්යාල වල වැඩ කටයුතු අන්තර්ජාලය හරහා නිවසේ සිට තම ආයතනයට සේවය සැපයීම අපට අසන්නට දකින්නට ලැබෙන දෙයකි. මෙහෙම අන්තර්ජාලය භාවිතා කරමින් අපි වැඩ කටයුතු සිදු කරන විට එහි ආරක්ෂාව පිළිබඳ වැඩි වශයෙන් සළකා බලන්න ඕනේ. කෙටියෙන්ම කිවහොත් Cyber Security කියන්නේ, අන්තර්ජාලයට සම්බන්ධ වී ඇති ඕනෑම පරිගණකයක හෝ ජංගම දුරකථනයක එය භාවිතා කළ යුතු අයිතියක් ඇති අයට හැර අන් අයට එම උපකරණ හරහා එහි ඇති දත්ත ලබාගත නොහැකි වන සේ ආරක්ෂා කිරීමයි.

මේ අය තමයි කලින් මම කියපු අනවසර ක්‍රියා සිදු කරමින් මුදල් සොරා ගැනීම, බැංකු පද්ධති වැනි මූල්‍ය අංශයේ පරිගණක පද්ධති වලට ප්‍රහාර එල්ල කිරීම සිදු කරන්නේ. මෙයාලට RedHat Hackers කියලත් කියනවා. මේ අය සම්පූර්ණයෙන්ම සිදු කරන්නේ නීති විරෝධී ක්‍රියා මගින් හානි පැමිණවීම.

කවුද මේ WhiteHat Hackers කියන්නේ?

මේ අය තමයි Ethical Hackers කියන්නේ. ඒ කියන්නේ Cyber Crimes වලදී පරිගනක පද්ධති ආරක්ෂා කරන්නේ මේ අය. Security Testing Purposes වලට මේ අය අදාළ අංශ වල අවසරය මත එම අදාළ පද්ධති වලට ඇතුල් වෙනවා. ලෝකය පුරාම Ethical Hackers ලා කියන්නේ IT Field එකේ ලොකු පිළිගැනීමක් තියෙන රැකියාවක්. ලෝකය පුරාම විවිධ ආයතන වල මේ අය සේවය කරනවා.

කවුද මේ GreyHat Hackers කියන්නේ?

මේ අය ටිකක් වෙනස් Hackers පිරිසක්. විටෙක හොඳ දේවල් වලට Hacking සිදු කරනවා වගේම තවත් වෙලාවට BlackHat Hackers ලා ගේ වැඩත් කරනවා. ඒ කියන්නේ වෙලාවකට නීති විරෝධී වැඩත් කරන පිරිසක්.

මොකද්ද මේ Social Engineering කියන්නේ?

මේක සරලව කිව්වොත් මිනිහෙක් Hacker කෙනෙකුට අවශ්‍ය දේ කරවා ගැනීමට පොළඹවා ගැනීම කියලා කියන්න පුළුවන්. ඒ කියන්නේ ඔයාගෙ ඕනේ එපාකම්, කන්න කැමති කෑම, දැනුම වගේ ගොඩක් දේවල් දීර්ඝ කාලයක් Analyse කරලා සමහර විට ඔයාගෙ ළඟම යාලුවෙක් වෙලා යම් යම් සංවේදී තොරතුරු ලබාගන්න එකට තමයි මේ Social Engineering කියන්නේ.

මොකද්ද මේ Phishing කියන්නේ?

සරලව කිව්වොත් Phishing කියන්නේ යම් කිසි Account එකකට Sign In වෙන Login Page එකක interface එකට සමාන වෙනත් Interface එකක් හදල ඒක අදාළ පුද්ගලයාට Social Engineering වැනි methods භාවිතා කරලා අදාළ පුද්ගලයාගේ සැබෑ Account එකේ Credentials ඇතුළත් කරවීමට සැලසුම් කිරීමයි. මෙහිදී එම පුද්ගලයා සැබෑ Account Login Interface එකකට සමාන පෙනුමක් දකින නිසා සැකයක් ඇති වීම අවමය. මෙසේ ඇතුළත් කරන තොරතුරු පහසුවෙන්ම අදාල ක්‍රියාව සිදු කළ පුද්ගලයාට ලබා ගත හැකිය. මේවායින් වැළකී සිටීමට ඔබ නොදන්නා E-Mails හා Links Open කිරීමෙන් වැළකී සිටිය යුතුය.

මොකද්ද මේ Computer Viruses & Trojan කියන්නේ?

ඔයාලා හැම කෙනෙක්ම අහල ඇති මේ Computer Virus කියන දේ. මේකෙදි වෙන්නෙ Crime එක සිදු කරන පුද්ගලයා විසින් Program කරන ලද Script එකක් E-Mail, Softwares හෝ වෙනත් ක්‍රම මගින් ඔබට එය ලැබීමට සැලසුම් කිරීමයි. එය Open කළ විට එහිදී ඔබේ පරිගනක පද්ධතියේ ක්‍රියාකාරීත්වය අකර්මණ්‍ය කර දැමීමට හෝ තොරතුරු විනාශ කර දැමීම හෝ අදාළ ක්‍රියාව සිදු කළ පුද්ගලයා Program කළ ආකාරයට එය Affect වෙයි. නොදන්නා Links, Crack Softwares, Crack Applications ඔබේ පද්ධතියට Install කරගැනීමේදී ඉතා සැලකිලිමත් විය යුතුය. එසේම Trajan ලෙස හදුන්වන්නේ ද මෙසේ පද්ධතියක් තුළට ඇතුළු වූ පසු එම පද්ධතියේ ක්‍රියාකාරීත්වය අදාළ Attacker ට Backdoor එකක් හරහා අවශ්‍ය තොරතුරු ලබා ගැනීම හා බොහෝ දේ සිදු කළ හැකි Malware එකක් හෙවත් අනිෂ්ට මෘදුකාංගයකි.

මොකද්ද මේ DoS & DDoS කියන්නේ?

  • DoS - Denial-Of-Service
  • DDoS - Distributed Denial-Of-Service

අපට අන්තර්ජාලයෙන් යම් තොරතුරක් සොයා ගැනීමට අවශ්‍ය වූ විට අපි ඔබ සියලු දෙනා යම් යම් Web අඩවි භාවිතා කරයි. මෙසේ අපට අවශ්‍ය දේ ලබා ගැනීමේදී අදාළ Server එකට Request යැවීමක් සාමාන්‍යයෙන් සිදු වෙයි. නමුත් DoS හා DDoS Attacks වලදී සිදු වන්නේ ඉතා විශාල Requests ප්‍රමාණයක් අදාළ Server එකට යැවීම මගින් එම Server එක තාවකාලිකව හෝ සම්පූර්ණයෙන්ම බිඳ වැටීමට සැළැස්වීමයි. මෙහිදී විශාල පිරිසක් භාවිතා කරන අත්‍යවශ්‍ය සේවා සම්පාදක පද්ධතියකට මෙසේ සිදු වුවහොත් පද්ධතිය යථා තත්ත්වයට පත් කරන තෙක් එය භාවිතා කිරීමට නොහැකි වෙයි.

මොකද්ද මේ Identity Theft කියන්නේ?

සරලවම කියනවා නම් Identity Theft කියන්නේ කෙනෙකුගෙ, නාමයන්, Credit Card පිළිබඳ තොරතුරු, බැංකු ගිණුම් පිළිබඳ තොරතුරු වැනි තොරතුරු ලබාගෙන එම තොරතුරු නීති විරෝධි ක්‍රියා වලදී භාවිතා කිරීමයි.

මොකද්ද මේ Ransomware කියන්නේ?

පුද්ගලයකු අන්තර්ජාලය හරහා විවිධ Crack Softwares, Applications තම පරිගනකයට ස්‌ථාපනය (Install) කරන විට එම Files තුළ මෙම Ransomware ඇතුලත්ව තිබී පරිගනක පද්ධතියට හානි ඇති විය හැකිය. මෙහිදී සිදුවන්නේ පරිගනකය තුළ ඇති සියලු ලිපිගොනු, Videos, Images, Audios, Other Documents බොහොමයක් Encrypt වීමයි. මෙසේ Encrypt වීම මගින් එය කියවීමට නොහැකි වෙයි. එම Fils ආපසු ලබා ගැනීමට නම් යම් කාල සීමාවක් තුළ එම Ransomware එක නිර්මාණය කර ඇති පුද්ගලයාට ගෙවීමක් කළ යුතු යැයි පරිගනකය පරිශිලනය කරන පුද්ගලයාට පරිගනකය හරහා දැකගැනීමට සලසයි. මෙම ගෙවීම් කටයුතු සිදු වන්නේ බොහෝ විට Crypto Currency හරහා වන නිසා මෙම විද්යුත් මුදල් සංසරණය වන්නේ කුමන පුද්ගලයාට ද යන්න සොයා ගැනීම අසීරු වෙයි. කෙසේ නමුත් මෙම ගෙවීම සිදු කර Attack එක සිදු කළ පුද්ගලයා නැවත Decryption Key එක ලබා දෙයි කියා 100% විශ්වාස කළ නොහැක.
